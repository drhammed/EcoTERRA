# Reproducibility Checklist

> Verify that a research project can be fully reproduced from raw data to final outputs.

---

## Data

- [ ] Raw data preserved in `Data/raw/` and never modified
- [ ] Data provenance documented (source, DOI, download date, query parameters)
- [ ] Data archived in public repository (Dryad, Zenodo, or Figshare) with DOI
- [ ] Data license specified
- [ ] Sensitive species locations handled (jittered or restricted)
- [ ] Column descriptions and units documented
- [ ] CRS documented for all spatial data

## Code

- [ ] All analysis scripts in version control (Git)
- [ ] Scripts run from project root without modification
- [ ] No hardcoded absolute paths (`here()` in R, relative paths in Python)
- [ ] Random seeds set for all stochastic operations
- [ ] Package versions pinned (`renv.lock` for R, `requirements.txt` for Python)
- [ ] Session info included at the end of analysis scripts
- [ ] Code deposited in public repository with DOI (GitHub + Zenodo)

## Pipeline

- [ ] Clear script numbering shows execution order (`01_`, `02_`, `03_`)
- [ ] README with step-by-step reproduction instructions
- [ ] Pipeline tested end-to-end in a clean environment
- [ ] Intermediate outputs saved and documented
- [ ] Model objects saved to `Models/` for inspection without re-running

## Manuscript

- [ ] All in-text statistics generated from code (not manually typed)
- [ ] Figures generated by scripts (not manually edited)
- [ ] Tables generated by scripts (not manually edited)
- [ ] Inline R/Python code used in Quarto/Rmarkdown (where applicable)
- [ ] Data Availability Statement includes archive DOIs
- [ ] Code Availability Statement includes repository URL and DOI

## Environment

- [ ] R version documented
- [ ] Python version documented (if applicable)
- [ ] Operating system noted
- [ ] Key package versions listed
- [ ] `renv.lock` or `requirements.txt` included in repository
- [ ] Docker/Singularity container provided (for complex environments — optional)

---

**Score**: Count passed items / total items × 100 = [X]%

| Score | Assessment |
|---|---|
| ≥ 90% | Fully reproducible |
| 75–89% | Mostly reproducible — minor gaps |
| 50–74% | Partially reproducible — significant gaps |
| < 50% | Not reproducible — major remediation needed |
